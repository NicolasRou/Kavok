# Kavok — Venture Studio & Software Factory

Landing page corporativa de Kavok. Construida con Next.js 16, Tailwind CSS v4 y TypeScript.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Stack

- **Next.js 16** (App Router)
- **Tailwind CSS v4**
- **TypeScript**
- **Plus Jakarta Sans** (via next/font)

## Deploy

Deploy on [Vercel](https://vercel.com).
